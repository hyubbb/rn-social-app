# 추가 기능
새로운 알림에 읽지 않았을때와 읽었을때의 DB칼럼 있어야할듯
첫페이지가 불러질때, checked의 값이 없는 쿼리의 전체값을 불러와서 화면에 보여준다. 
알림창을 눌렀을때 userId를 체크하여 checked의 값이 false인 값들을 전부다 true 로 변경하여 읽음처리를 할것.
